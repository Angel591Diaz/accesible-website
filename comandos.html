Principales comandos de Git

Git ofrece una variedad de comandos para gestionar proyectos de desarrollo. A continuación se presentan los más usados:

  • git init: Inicializa un nuevo repositorio Git en el directorio actual.
  • git clone [url]: Clona un repositorio remoto en tu máquina local.
  • git status: Muestra el estado actual del repositorio, incluyendo archivos modificados y sin seguimiento.
  • git add [archivo]: Añade archivos al área de preparación (staging area).
  • git commit -m "mensaje": Crea un commit con los cambios preparados y un mensaje descriptivo.
  • git log: Muestra el historial de commits del repositorio.
  • git branch: Lista, crea o elimina ramas (branches).
  • git checkout [rama]: Cambia a otra rama existente.
  • git merge [rama]: Fusiona la rama indicada con la rama actual.
  • git pull: Descarga y fusiona los cambios del repositorio remoto en la rama actual.
  • git push: Sube los cambios locales al repositorio remoto.
